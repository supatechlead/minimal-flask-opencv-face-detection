### Minimal Flask face detection using OpenCV Haarcascades

Install Python Flask and OpenCV using ```pip install flask``` and ```pip install opencv-python```.\
Run with ```python app.py```, then open flask app on your browser.

Local virtual environment may be created using ```python -m venv venv``` then activated with ```source venv/bin/activate```.\
Modules may be install from ```pip install requirements.txt```.
